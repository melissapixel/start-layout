# 🎯 План обучения HTML/CSS → JavaScript Pro

### Темп:
Не привязываемся к неделям. Каждый этап — до полного понимания.

---

## 🗺️ Дорожная карта

### <ins>Этап 1. Физика браузера</ins> ✅
- Что такое Document Flow
- Блочная модель, margin collapse
- Float и его побочные эффекты
- **Результат:** понимаешь, почему элементы встают так, а не иначе

### <ins>Этап 2. Каскад и специфичность</ins>
- Как браузер решает, чей стиль победит
- Наследование и его границы
- **Результат:** CSS не ломается при добавлении новых стилей

### <ins>Этап 3. Flexbox</ins>
- Одна ось, гибкость, выравнивание
- **Результат:** 90% современных макетов за минуты

### <ins>Этап 4. Grid</ins>
- Двумерная раскладка, области, auto-placement
- **Результат:** сложные лейауты собираются декларативно

### <ins>Этап 5. Position и Stacking Context</ins>
- z-index, слои, контексты наложения
- **Результат:** модалки, дропдауны, тултипы без боли

### <ins>Этап 6. Адаптив и юниты</ins>
- rem/em/vw/vh, media queries, mobile-first
- **Результат:** сайт работает на любом устройстве

### <ins>Этап 7. Production-архитектура</ins>
- БЭМ, CSS-переменные, методология ITCSS
- **Результат:** CSS масштабируется и не превращается в ад

### <ins>Этап 8. Мостик в JavaScript</ins>
- Как CSS влияет на reflow/repaint
- Управление состоянием через классы
- Data-атрибуты как хуки для JS
- **Результат:** верстаешь так, чтобы JS-разработчик не проклинал тебя


> 💡 **Production-совет:** В обучении порядок важнее скорости. Пропустишь Flow — будешь всю жизнь гуглить "почему мой flex не работает".
